## 1.下载
```bash
git clone https://gitee.com/sofunmanager/sofun-chat-gpt.git
```
## 2.进入项目根目录
```bash
cd sofun-chat-gpt
```

## 3.安装依赖
 
```bash
npm install
```

## 4.启动（可选，不开发的情况无需调试）

```bash
npm run dev
```

## 5.编译

```bash
# windows
npm run build:win

# macOS
npm run build:mac

# Linux
npm run build:linux
```

## 6.编译后的安装包路径为
```
根目录/dist/...
```
